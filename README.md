# Python Web Calculator

## 📌 Project Overview

This project is a **web-based calculator application** developed using **Python (Flask)** for the backend and **HTML, CSS, and JavaScript** for the frontend.
The calculator provides an interactive user interface that allows users to perform basic arithmetic operations directly in the browser.

The project demonstrates how Python can be integrated with web technologies to build simple and functional web applications.

---

## 🚀 Features

* Perform basic arithmetic operations:

  * Addition
  * Subtraction
  * Multiplication
  * Division
* Interactive calculator button interface
* Responsive and user-friendly UI
* Real-time calculation results
* Clean project structure using Flask framework

---

## 🛠 Technologies Used

## ⚙️ Installation and Setup

1. **Clone the repository**

```
git clone https://github.com/sravya0125/python-web-calculator.git
```

2. **Navigate to the project directory**

```
cd python-web-calculator
```

3. **Install required dependency**

```
pip install flask
```

4. **Run the application**

```
python app.py
```

5. **Open in browser**

```
http://127.0.0.1:5000
```
